TeqCMS: managing a site through context and an agent

TeqCMS is a CMS for developers, not editors. I define tasks in context, the agent edits files, and git tracks every change.

  • no browser UI
  • no database
  • git and AI ensure full control
AISM architecture: from context to site

How it works

  1. You write a task or context
  2. The agent creates HTML
  3. You accept the result
  4. AI translates into all locales
