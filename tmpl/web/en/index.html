Tequila CMS — minimalistic AI-oriented CMS without databases or admin panels

TeqCMS is a way to manage a multilingual website as code. Content is stored in files, edited via git and automatically translated using LLMs.

  • ⏱ Quick start: deployment without builds or databases
  • 🌍 AI localization: translations update themselves
  • 💾 Transparency: the entire site consists of regular HTML files
  • 🔌 SSR + templates: Nunjucks, Mustache or plain HTML
  • 🤖 LLM-assisted development: page-level prompt support

TeqCMS is perfect for landing pages, documentation, SaaS promotions and indie projects with international audiences.
