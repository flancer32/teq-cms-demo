TeqCMS: управление сайтом через контекст и агента

TeqCMS — это CMS для разработчиков, а не для редакторов. Я описываю задачи в контексте, агент правит файлы, а git хранит историю.

  • без пользовательского UI
  • без базы данных
  • git и AI обеспечивают полный контроль
Архитектура AISM: от контекста к сайту

Как это работает

  1. Пишешь задачу или контекст
  2. Агент создаёт HTML
  3. Ты принимаешь результат
  4. AI переводит на все локали

Кому подойдёт

Подход рассчитан на тех, кто ценит контроль и автоматизацию.

  • Если вы пишете HTML вручную
  • Если вы работаете через git
  • Если вам нужен AI, но не хотите городить UI

Примеры использования

  • Вы создаёте лендинг — агент пишет всё сразу
  • Вы ведёте документацию — AI обновляет переводы
  • Вы развёртываете сайт — просто git clone + node
