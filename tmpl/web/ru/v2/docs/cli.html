CLI и автоматизация

TeqCMS поставляется с набором CLI-команд для запуска сервера и управления переводами.

  • npx teq-cms web — запуск веб-сервера
  • npm run translate — обновление переводов

Скрипты можно расширять и дополнять через плагины или собственные npm-команды.
