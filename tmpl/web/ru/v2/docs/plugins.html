Подключаемые модули

TeqCMS расширяется за счёт модулей, подключаемых в DI‑контейнер. Любой компонент можно заменить без изменения ядра.

  • создание собственных адаптеров шаблонов и данных
  • подмена логики перевода или хранения
  • добавление новых команд CLI

Гибкая архитектура позволяет встроить CMS в существующие проекты или написать собственные плагины.
