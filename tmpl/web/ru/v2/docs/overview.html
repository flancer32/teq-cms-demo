Обзор архитектуры

TeqCMS построена вокруг DI-контейнера @teqfw/di и модульного подхода. Все страницы и шаблоны лежат в файловой системе и управляются через модули.

  • Шаблоны: tmpl/web/{locale}/
  • Статика: web/
  • Служебные файлы CMS: var/teq-cms/

CMS не требует базы данных и может работать с любым Node.js-сервером.
