О проекте

@flancer32/teq-cms-promo развёрнут на движке TeqCMS как отдельный плагин. Это полноценный продукт, работающий как обычное SSR‑приложение без сборок и баз данных.

Что это за сайт

  • рабочий проект, построенный на TeqCMS
  • реализован в виде плагина CMS
  • хостится без frontend‑фреймворков и БД

Назначение сайта

  • демонстрация возможностей CMS в реальном контексте
  • стартовая точка для разработчиков file‑based подхода
  • площадка для тестов AI‑генерации и автолокализации

Как устроен

  • страницы хранятся в Git как HTML‑шаблоны
  • генерируются через LLM по инструкциям из teq-cms-promo-ctx
  • локализация выполняется встроенным AI‑механизмом

Для кого

  • инженерам, которые хотят писать сайты как код
  • командам, ценящим контроль и минимальную инфраструктуру
