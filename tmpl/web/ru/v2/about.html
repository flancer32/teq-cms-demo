О проекте

@flancer32/teq-cms-promo развёрнут на движке TeqCMS и демонстрирует реальные возможности системы. Это полноценный веб‑сайт, работающий как обычное SSR‑приложение без сборок и баз данных.

Что это за сайт

  • полноценный веб‑сайт на TeqCMS, а не просто демо
  • nodejs‑приложение с SSR‑страницами на шаблонизаторе Nunjucks
  • хостится без сборок, БД и frontend‑фреймворков

Назначение сайта

  • демонстрация возможностей TeqCMS в реальном контексте
  • стартовая точка для разработчиков file‑based CMS
  • площадка для тестирования AI‑генерации и автолокализации

Как устроен

  • все страницы хранятся в Git как Nunjucks‑шаблоны
  • генерируются LLM по инструкциям из приватного каталога teq-cms-promo-ctx
  • локализация выполняется вручную через встроенный AI‑механизм CMS

Для кого

  • Индихакеры, которым нужен минималистичный и расширяемый движок
  • Технические основатели, желающие быстро развернуть проект с ростом
  • Команды, использующие LLM, включая автоагентов и Codex‑пайплайны
  • Разработчики модульных систем, ценящие гибкость и контроль без скрытой логики
